# MAC Address Changer

A simple Python tool to change the MAC address of a network interface on Linux systems using `macchanger`.

---

## Features

- Easy to use
- Changes MAC address instantly
- Automatically disables and enables the interface
- Beginner-friendly Python project
- Error handling included

---

## Requirements

- Python 3
- Linux OS
- `macchanger`

Install `macchanger`:

```bash
sudo apt install macchanger
```

---

## Usage

```bash
python3 mac_changer.py
```

Example:

```bash
Enter new MAC address : 00:11:22:33:44:55
Enter interface name : wlan0
```
